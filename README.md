# 🎓 Calculadora de Média Escolar Pro

![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
![CustomTkinter](https://img.shields.io/badge/UI-customtkinter-464646?logo=Python)
![License](https://img.shields.io/badge/license-MIT-blue)

Uma aplicação desktop moderna desenvolvida em Python para facilitar o controle de notas estudantis. O sistema calcula médias automaticamente e permite exportar os dados diretamente para arquivos Excel.

## ✨ Funcionalidades

- **Interface Moderna:** UI personalizada com CustomTkinter (Modo Light).
- **Cálculo em Tempo Real:** Processamento automático de médias com status de Aprovação/Reprovação.
- **Exportação para Excel:** Salva Nome, Notas, Média e Resultado em arquivos `.xlsx`.
- **Banco de Dados em Planilha:** Se o arquivo Excel já existir, o app adiciona o novo aluno sem apagar os dados anteriores.
- **Janela Customizada:** Barra de título personalizada (sem bordas padrão do Windows).

## 🚀 Como Executar

### Pré-requisitos
Você precisará do Python 3.x instalado e das seguintes bibliotecas:
- `customtkinter`
- `pandas`
- `openpyxl`
